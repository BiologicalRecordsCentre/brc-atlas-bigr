Module: src/projections

(constant) module:src/projections :array

This object describes the coordinate reference systems used in this project corresponding to the British National Grid, Irish Grid, UTM zone 30N (used for the Channel Islands) and WGS 84. The object contains four properties, each named with the two letter code used throughout this package to represent one of the three systems: gb, ir, ci and wg. Each of these properties provides access to an object defining the name, epsg code and proj4 string for the CRS.
