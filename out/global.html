widthFromHeight(transOpts, outputHeight)

Given a transform object, describing a bounding rectangle in world coordinates, and a height dimension, this function returns a width dimension that respects the aspect ratio described by the bounding rectangle.
Parameters:
Name Type Description
transOpts object the transformation object
outputHeight number the height, e.g. height in pixels, of an SVG element.
